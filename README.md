# Advanced GUI Calculator

## 📌 Description
A scientific calculator built using Python and Tkinter with a user-friendly graphical interface.  
It performs both basic arithmetic operations and advanced mathematical functions.

---

## 🚀 Features
- Basic operations: Addition, Subtraction, Multiplication, Division  
- Scientific functions: sin, cos, square root  
- Keyboard input support  
- Error handling for invalid inputs  
- Clean and interactive dark-themed UI  

---

## 🛠️ Technologies Used
- Python  
- Tkinter  
- Math module
